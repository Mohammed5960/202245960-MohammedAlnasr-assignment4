# Personal Portfolio Web Application – Assignment 4

## Project Overview
This project is the final version of my personal portfolio web application. It combines all the features developed in previous assignments and improves them into a more polished and professional product.

The goal of this project is to demonstrate my ability to build an interactive web application using HTML, CSS, and JavaScript, while applying concepts such as API integration, state management, and user experience design.

## Features

### 1. Interactive Project Section
- Search projects dynamically
- Filter by category (Web, JavaScript, Form)
- Sort projects (A–Z, Z–A, Newest, Oldest)

### 2. GitHub API Integration
- Fetches public repositories from GitHub
- Displays:
  - Repository name
  - Description
  - Language
  - Star count
- Handles loading and error states

### 3. State Management
- Dark/light mode saved using localStorage
- Visitor name is saved and displayed on return

### 4. Time on Site Feature (Innovation)
- Displays how long the user has been on the website
- Updates every second dynamically

### 5. Advanced Contact Form
- Validates:
  - Name (minimum 2 characters)
  - Email (valid format)
  - Message (minimum 10 characters)
- Shows clear error messages and success confirmation

### 6. User Experience
- Smooth scrolling and transitions
- Responsive design for mobile and desktop
- Clear instructions for user interaction
