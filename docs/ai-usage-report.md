# AI Usage Report

## 1. Tools Used and Use Cases

During this assignment, I used **ChatGPT** as an AI tool to assist in the development process.

The main use cases included:

- Helping generate and improve JavaScript code
- Debugging errors and fixing issues in logic
- Suggesting UI/UX improvements (layout, spacing, user guidance)
- Improving documentation structure and clarity
- Assisting in writing README and technical documentation

ChatGPT was mainly used as a support tool to guide development and improve code quality, not to fully generate the project.

---

## 2. Benefits and Challenges

### Benefits:
- Helped solve problems faster when errors occurred
- Provided clear explanations for JavaScript logic
- Suggested better ways to structure the code
- Improved the overall quality of documentation
- Saved time during debugging and testing

### Challenges:
- Some suggestions were too generic and needed adjustment
- Occasionally required rewriting to match assignment requirements
- Needed careful review to ensure correctness

---

## 3. Learning Outcomes

Using AI tools helped me improve in several areas:

- Better understanding of JavaScript concepts (DOM, events, API)
- Improved debugging skills
- Learned how to structure code more clearly
- Gained experience in writing technical documentation
- Learned how to combine multiple features into one application

Overall, AI helped me learn faster but I still needed to understand and apply the concepts myself.

---

## 4. Responsible Use and Modifications

All AI-generated suggestions were carefully reviewed and modified.

I ensured that:
- The code matches assignment requirements
- The logic is fully understood before using it
- The final work is my own implementation
- Changes were made to improve clarity and originality

AI was used as a support tool, not as a replacement for learning or development.

---

## 5. Conclusion

AI tools like ChatGPT were helpful in improving both the development process and the final quality of the project. However, understanding the concepts and making modifications was essential to ensure correctness and originality.
